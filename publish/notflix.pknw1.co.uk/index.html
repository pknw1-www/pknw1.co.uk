Private Access Streaming & Services

The Private Streaming Services provide users with access to media from all streaming services in a secure and anonymous fashion using Plex Media Server
Users can self-add any media by using the requests interface which grabs the metadata and then searches the web for copies 

Also,  recently added, users can submit any YouTube url and the video will be downloaded and added to Plex with no adverts or breaks
